Orbital Keybindings

• Super refers to your MOD key, typically referred to as your Windows key.

Function Shortcut
Shortcut Menu Super + H
Close Window Super + Shift + Q
Application Launcher (Rofi) Super + D
Terminal Emulator Super + Return
Switch forwards to window Super + Left
Switch backwards to window Super + Right
Move window forwards Super + Shift + Left
Move window backwards Super + Shift + Right
Toggle fullscreen Super + Shift + F
Switch workspaces Super + 0-9
Toggle workspaces Super + Tab
Cycle workspaces forward Super + Alt + .
Cycle workspaces backward Super + Alt + ,
Make windows larger (vertically) Super + Alt + Up
Make windows smaller (vertically) Super + Alt + Down
Make windows larger (horizontally) Super + Alt + Right
Make windows smaller (vertically) Super + Alt + Left
Log out of Orbital Super + Shift + E
